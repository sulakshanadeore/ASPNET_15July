   Install EF

2) In the Class Library
   Add Reference System.ComponentModel.DataAnnotations

using System.ComponentModel.DataAnnotations;
using System.ComponentModel.DataAnnotations.Schema;

  A) create (Entity) Classes as per the requirement-
---------------------------------------------------------------
	The Classes will contain only properties
 Each Property will have a Attribute given to it:
	1) Required
	2) Key
	3) Column-- Order,Datatype etc
	4) Foreignkey
	5) Table
	6) NotMapped
        7) Display
	8) Scaffold
	9) MinLength
	10) MaxLength
	11)DataType

  public class Employee
 {

 }


Table will be created as "Employees"
